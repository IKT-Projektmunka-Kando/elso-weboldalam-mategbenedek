Branch: Nyiss egy új ágat a saját neveddel (pl. feature/milan).

Kép feltöltés:

Keress/készíts magadról (vagy egy avatárról) egy képet.

Nevezd el ékezetek nélkül (pl. milan.jpg).

Másold be az img mappába.

Kódolás (HTML):

Nyisd meg az index.html-t.

KERESD MEG A SAJÁT NEVEDET! (Ne nyúlj máséhoz!)

A saját <img> címkédben írd át az src="img/*****.jpg"-t a saját képed nevére (pl. src="img/milan.jpg").

A saját <a> címkédben a href="#" helyére másold be a saját GitHub profilod vagy egy másik repód linkjét (pl. href="https://sandornefr.github.io/9.C_1.cs_web/").

Mentés és Beküldés:

Commitold a változásokat ("Saját kártya frissítése").

Pushold a Branch-et.
Nyiss egy Pull Requestet.
